Following are the curl request for accessing todo list. The values given are just examples, which needs to replaced by real values.

1) Creates a todo item under a specific todo list
curl -X 'POST' \
  'http://localhost:3000/todo_lists/{todo_list_id}/todo_items' \
  -d '{
  "todo_item": {
    "title": "string",
    "description": "string",
    "flagged": true,
    "parent_id": null,
    "schedule_id": null
  }
}'

2) Creates a todo item under a specific todo item. Not more than 1 level is supported
curl -X 'POST' \
  'http://localhost:3000/todo_lists/{todo_list_id}/todo_items' \
  -d '{
  "todo_item": {
    "title": "string",
    "description": "string",
    "flagged": true,
    "parent_id": 1, # Parent todo item id, for subtask. Else null.
    "schedule_id": null
  }
}'

3) Flag a todo list
curl -X 'PATCH' \
  'http://localhost:3000/todo_lists/{todo_list_id}/todo_items/{todo_item_id}' \
  -H 'accept: */*' \
  -H 'Content-Type: application/json' \
  -d '{
  "todo_item": {
    "flagged": true # false to unflag it
  }
}'

4) Completes a todo item
curl -X 'PATCH' \
  'http://localhost:3000/todo_lists/{todo_list_id}/todo_items/{todo_item_id}' \
  -H 'accept: */*' \
  -H 'Content-Type: application/json' \
  -d '{
  "todo_item": {
    "completed": "true" # false to make the item incomplete
  }
}'
